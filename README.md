# Private Telegram ChatGPT bot
### Settings
Set up **settings.json** file:

```json
{
  "bot": "YOUR_BOT_API_KEY",
  "gpt": {
    "model": "gpt-3.5-turbo",
    "key": "YOUR_OPENAI_API_KEY"
  },
  "allowedUsers": []
}
```
